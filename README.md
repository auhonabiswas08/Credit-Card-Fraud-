• Objective  
  To develop a credit card fraud detection model using transaction pattern analysis to enhance financial security.

• Approach  
  Applied Decision Trees for high interpretability and improved transparency in fraud detection.  
  Enhanced robustness by using an ensemble of decision pathways.  
  Evaluated model performance using Confusion Matrix, F1-score, and AUC-ROC metrics.

• Results  
  Achieved 96.4% accuracy, indicating strong predictive performance.  
  Recorded a 96.3% AUC-ROC score, demonstrating excellent class separability.
